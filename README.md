# qwen-landing

Next.js (App Router) приложение для генерации паспортов продуктов по методике «Полярная звезда».

## Supabase health-check

Для корректной работы эндпоинта `/api/health-supabase` необходимо:

1. Создать таблицу `public._health` и добавить в неё хотя бы одну строку (любой dummy-ID).
2. Выдать `SELECT`-политику для анонимного ключа **или** оставить доступ только через service role.

Эндпоинт сначала пытается подключиться с `SUPABASE_SERVICE_ROLE_KEY` (если задан), затем с анонимным ключом.

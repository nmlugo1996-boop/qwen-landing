## 🧠 КОГНИТИВНЫЙ БЛОК

### 1.a. Изменение модели потребления
**Новый рынок:** «Мясной протеин без компромиссов» — заменяем:
1. Протеиновые батончики/шейки (воспринимаются как «неестественные»).  
2. Домашнюю готовку мяса (время, посуда, уборка).  
3. Традиционные мясные снеки (пахнут, пачкают, неудобны).

**Доп. монетизируемая ценность:**
- «Настоящее мясо в умном формате» — премиум за аутентичность + удобство.  
- «Белок без чувства вины» — премиум за чистоту состава.  
- «Топливо для результата» — премиум за эффективность и скорость усвоения.

_Source / Evidence / Assumptions_: бриф + рыночные референсы (Quest, RXBAR, ЭкоНива Protein). Метрики проверки: WTP-опросы, конверсия сэмпл→покупка.

---

### 1.b. Изменение технологии потребления: новые ритуалы

| Старый ритуал | Новый ритуал |
|---|---|
| Готовить мясо после тренировки | Достать снек из сумки и съесть сразу в раздевалке |
| Пить протеиновый шейк «через не хочу» | Наслаждаться вкусом настоящего мяса в удобном формате |
| Прятать еду из-за запаха | Есть демонстративно — маркер экспертности |
| Есть в одиночку | Делится с партнёром (формат «2 в упаковке») |

**Новые привычки:**
1. «Пост-тренировочный ритуал»: 1 снек + вода сразу после зала.  
2. «За рулём без стресса»: есть одной рукой, без крошек/запаха.  
3. «Демонстрация экспертности»: достал МЯСОВОЙ КОД — ты в теме.

_Source / Evidence / Assumptions_: бриф, примеры Red Bull / Kind Bar.

---

### 1.c. Нарративы (формат «Когда вы… то…»)

**Короткий (headline):** «Когда вы едите МЯСОВОЙ КОД после тренировки — вы получаете 20 г чистого животного белка без запаха и тяжести.»  
**Средний:** «Когда вы выбираете МЯСОВОЙ КОД вместо протеинового батончика, вы кормите тело тем, что оно эволюционно создано усваивать — настоящим мясом в формате, который работает на ваш ритм жизни.»  
**Длинный (story):** «Ты потеешь в зале. Ты ставишь рекорды… МЯСОВОЙ КОД — настоящее мясо для тех, кто строит результат. Без запаха. Без грязи. Белок. Без компромиссов.»

_Use_: headline — упаковка; medium — лендинг; long — PR/бренд материалы.

## 👁 СЕНСОРНЫЙ БЛОК

> Для каждой модальности — описание желаемого опыта, измеримая цель, протокол теста, метод анализа, критерий приёмки и технические рекомендации.

### 2.1 Визуальный образ
**Желаемый опыт:** Матовый графит + неоновый акцент; крупная маркировка «20г».  
**Измеримая цель:** премиальность ≥7/9 (N=50).  
**Протокол:** слепой показ макетов (A/B/C), нейтральное освещение, анкета.  
**Критерий приёмки:** среднее ≥7; ≤15% оценок ≤4.  
**Рекомендации:** матовая ламинация, контрастный шрифт.

_Source / Evidence / Assumptions_: целевые пороги — [ПРЕДПОЛОЖЕНИЕ].

---

### 2.2 Аудиальный образ
**Желаемый опыт:** лёгкий щелчок при открытии; упругий откус.  
**Измеримая цель:** sound satisfaction ≥7/9 (N=30).  
**Протокол:** in-use test (реальная упаковка), запись звука, анкета.  
**Критерий приёмки:** ≥90% отсутствие жалоб на крошение; среднее ≥7.  
**Рекомендации:** запатентованный клик-механизм/seal, формула с низким крошением.

---

### 2.3 Обонятельный образ
**Желаемый опыт:** минимальная интенсивность неприятного запаха при открытии; легкие ноты специй при жевании.  
**Измеримая цель:** неприятный запах ≤2/9 через 0 и 5 минут (N=30), ≤10% оценок >3.  
**Протокол:** слепая дегустация; 0/5/15 мин; помещение 22±2°C.  
**Критерий приёмки:** среднее ≤2; ≤10% >3.  
**Рекомендации:** энзимная обработка, микрокапсулирование, N₂-flush и барьерная упаковка (OTR низкий) [ПРЕДПОЛОЖЕНИЕ].

---

### 2.4 Осязательный образ
**Желаемый опыт:** soft-touch упаковка, продукт не липнет.  
**Измеримая цель:** no-mess: потеря массы на пальцах <0.05 г (N=20).  
**Протокол:** контактный тест, температура 22±2°C, влажность 45–55%.  
**Критерий приёмки:** средняя потеря <0.05 г; ≤10% >0.05 г.  
**Рекомендации:** съедобная тонкая плёнка, покрытие продукта, soft-touch ламинация.

---

### 2.5 Вкусовой образ
**Желаемый опыт:** чистый мясной вкус, лёгкий финиш, линейка вкусов (классика, паприка, травы-лимон).  
**Измеримая цель:** общий вкус ≥7/9 (N=50); % тяжести <20%.  
**Протокол:** слепая дегустация 3 вкусов, профиль: 18–45, активные в зале.  
**Критерий приёмки:** средняя ≥7; тяжесть ≤20% случаев.  
**Рекомендации:** баланс соли/жира, натуральные специи, снижение жировой плёнки.

_Source / Evidence / Assumptions_: сенсорные пороги — [ПРЕДПОЛОЖЕНИЕ].

## Производство / Регуляторика / QA

**Технологический процесс (кратко):** приём сырья → предобработка → энзимная обработка (нейтрализация запаха) → смешивание → формование → термообработка (≥72°C) → сушка/контроль aw → фасовка с N₂-flush → контроль.  
**CCP:** сырьё, термообработка, aw, герметичность упаковки.  
**Shelf-life:** 180 дней (RT tests + accelerated), критерии: TPC, Salmonella, сенсорика.  
**Микробиология:** TPC, Salmonella absent in 25 g, Listeria absent.  
**Маркировка:** ингредиенты, пищевая ценность 100 g и порция, аллергены, дата.

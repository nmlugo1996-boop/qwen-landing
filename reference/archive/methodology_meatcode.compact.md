## 📋 КРАТКИЙ ПАСПОРТ
Параметр: Категория; Значение: мясные снеки для пост-тренировочного восстановления  
Названия (рабочие): МЯСОВОЙ КОД / BeefGo / PROTEIN CUT  
Аудитория: 18–45 лет, посетители спортзала, ценящие функциональное питание  
Боль: нужен быстрый качественный животный белок; существующие варианты — либо «химия», либо неудобство/запах/грязь  
Уникальность: компактный 40 г снек → 20 г белка; технология нейтрализации запаха; упаковка no-mess (есть одной рукой)  
Формат/показатели: 40 г / 20 г белка / ≈150 ккал [проверить] / РРЦ 149–179 ₽  
**One-liner:** МЯСОВОЙ КОД — настоящее мясо в формате компактного пост-тренировочного снека: 20 г белка, без запаха и без грязи.

## 1. КОГНИТИВНЫЙ БЛОК
### 1.a. Изменение модели потребления
**Новый рынок:** «Мясной протеин без компромиссов» — заменяет: протеиновые батончики/шейки, домашнюю готовку мяса, традиционные мясные снеки.  
**Доп. монетизация:** премиум за аутентичность и удобство; «Белок без чувства вины»; эффективность усвоения.

### 1.b. Технология потребления / ритуалы
Старый → новый: готовка после зала → достал снек и съел в раздевалке.  
Новые привычки: пост-тренировочный снек+вода; есть одной рукой в машине; демонстративное потребление как маркер экспертности.

### 1.c. Нарративы
Headline: «20 г чистого животного белка без запаха и тяжести.»  
Medium/Long: короткие сценарии для упаковки, лендинга и PR.

### 1.d. Модель убеждений и обучения
Мысли: «Я выбираю эффективность.» Поведение: демонстрация, рекомендация, подписка.  
Каналы обучения: 15-с ролики, упаковка-комикс, QR-гайды, амбассадоры, UGC.

## 2. СЕНСОРНЫЙ БЛОК (ключевые требования)
**2.1 Визуал:** матовый графит + неоновый акцент; крупная маркировка «20г».  
**2.2 Аудиал:** щелчок при открытии; упругий откус.  
**2.3 Обоняние:** минимальный неприятный запах при открытии; лёгкие ноты специй при жевании.  
**2.4 Тактильность:** soft-touch упаковка; no-mess — продукт не липнет.  
**2.5 Вкус:** чистый мясной профиль, лёгкий финиш; базовые вкусы: классика, паприка, травы-лимон.
Для каждой модальности — измеримые цели и простые протоколы (слепые тесты, N=30–50, критерии приёмки заданы).

## 5. ПРОИЗВОДСТВО / QA / РЕГУЛЯТОРИКА
Процесс: приём сырья → предобработка → энзимная обработка (нейтрализация запаха) → формование → термообработка (≥72°C) → сушка/контроль aw → фасовка N₂-flush.  
CCP: сырьё, термообработка, aw, герметичность. Shelf-life ≈ 180 дней (RT + accelerated). Микробиология: TPC, Salmonella absent, Listeria absent.

## 7. PLAN ВАЛИДАЦИИ (минимум)
1. Sensory panel (N=50) — smell ≤2, taste ≥7.  
2. No-mess test (N=20) — loss <0.05 g.  
3. Pilot в 5 клубах (N≥500) — sample→purchase ≥25%.  
4. A/B лендинг (10k traffic).  
5. Price elasticity test (129/149/179/199/249).  
6. Shelf-life real + accelerated.
